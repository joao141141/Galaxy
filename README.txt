
English:
In this project, I'll use a neural network to classify galaxies in deep space using image data from Galaxy Zoo. This project aims to automate and improve the accuracy of galaxy classification, contributing to scientific discovery.

Português:
Neste projeto, usarei uma rede neural para classificar galáxias no espaço profundo usando dados de imagem do Galaxy Zoo. Este projeto tem como objetivo automatizar e melhorar a precisão da classificação de galáxias, contribuindo para a descoberta científica.
